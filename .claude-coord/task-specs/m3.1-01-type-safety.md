## Common Type Error Fixes

### 1. LangGraph State Types
```python
# Problem: LangGraph uses dynamic state
from langgraph.graph import StateGraph
graph = StateGraph(WorkflowState)  # WorkflowState must be TypedDict

# Solution: Use TypedDict properly
from typing import TypedDict

class WorkflowState(TypedDict, total=False):
    stage_outputs: Dict[str, Any]
    current_stage: str
    # ... all fields optional
```

### 2. Pydantic Config Validation
```python
# Problem: Optional fields accessed without checking
def execute(config: StageConfig):
    agents = config.agents  # Might be None!

# Solution: Check or provide defaults
def execute(config: StageConfig):
    agents = config.agents or []
    # Or use Pydantic Field default
```

### 3. Registry Pattern Type Erasure
```python
# Problem: Registry loses type information
def get(name: str) -> Any:
    return self._registry[name]

# Solution: Use generics
T = TypeVar('T')
def get(name: str, type_: Type[T]) -> Optional[T]:
    item = self._registry.get(name)
    return item if isinstance(item, type_) else None
```

### 4. JSON Schema to Type
```python
# Problem: Dict[str, Any] everywhere
def load_config(path: str) -> Dict[str, Any]:
    return yaml.safe_load(...)

# Solution: Parse to typed dataclass
@dataclass
class WorkflowConfig:
    name: str
    stages: List[StageConfig]

def load_config(path: str) -> WorkflowConfig:
    data = yaml.safe_load(...)
    return dacite.from_dict(WorkflowConfig, data)
```
