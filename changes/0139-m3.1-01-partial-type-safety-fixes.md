## Technical Details

### Type Annotation Patterns Used

**1. Context Manager Return Types:**
```python
# Before
@contextmanager
def __call__(self):
    ...

# After
@contextmanager
def __call__(self) -> Generator[None, None, None]:
    ...
```

**2. Callable Type Parameters:**
```python
# Before
def __init__(self, state_getter: Optional[Callable] = None):
    ...

# After
def __init__(self, state_getter: Optional[Callable[[], Dict[str, Any]]] = None):
    ...
```

**3. Tuple Type Parameters:**
```python
# Before
history: Dict[tuple, List[float]] = {}

# After
history: Dict[tuple[str, str], List[float]] = {}
```

**4. Dict Value Type Safety:**
```python
# Before
if "command" in action:
    return action["command"]  # Returns Any

# After
if "command" in action:
    cmd = action["command"]
    return str(cmd) if cmd is not None else None  # Returns str | None
```

**5. Variable Type Annotations for Complex Dicts:**
```python
# Before
stats = {
    "cpu_time": None,
    "memory_delta": None,
    "cpu_exceeded": False
}  # mypy infers wrong type

# After
stats: Dict[str, Any] = {
    "cpu_time": None,
    "memory_delta": None,
    "cpu_exceeded": False
}  # explicit type
```
