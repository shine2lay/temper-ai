## Key Fixes This Session

### 1. **kwargs Type Annotations

**Problem:** Functions with `**kwargs` need explicit type hints for mypy --strict.

**Solution:**
```python
# Before
def execute(self, **kwargs) -> ToolResult:

# After
def execute(self, **kwargs: Any) -> ToolResult:
```

**Files:** calculator.py, web_scraper.py, file_writer.py, prompt_engine.py, service.py, llm_cache.py

### 2. Optional[] for Default None Parameters

**Problem:** Implicit Optional[] is not allowed in mypy strict mode.

**Solution:**
```python
# Before
def __init__(self, config: Dict[str, Any] = None):

# After
def __init__(self, config: Optional[Dict[str, Any]] = None):
```

**Files:** file_writer.py, web_scraper.py, backend.py

### 3. Type Parameters for Generic Types

**Problem:** Generic types like Dict, List, ContextManager need type parameters.

**Solution:**
```python
# Before
def get_session_context(self) -> ContextManager:
def test_sequential(...) -> Tuple[str, Dict]:

# After
def get_session_context(self) -> ContextManager[Any]:
def test_sequential(...) -> Tuple[str, Dict[str, Any]]:
```

**Files:** backend.py, sequential_testing.py

### 4. SQLAlchemy Column Method Type Ignores

**Problem:** SQLAlchemy columns have methods like `.isnot()` that mypy doesn't recognize on dict types.

**Solution:**
```python
# Before
query = select(WorkflowExecution).where(
    WorkflowExecution.extra_metadata.isnot(None)
)

# After
query = select(WorkflowExecution).where(
    WorkflowExecution.extra_metadata.isnot(None)  # type: ignore[union-attr]
)
```

**Files:** metrics_collector.py

### 5. Union Types for Network Lists

**Problem:** List of mixed IPv4Network and IPv6Network needs explicit Union type.

**Solution:**
```python
# Before
BLOCKED_NETWORKS = [
    ipaddress.IPv4Network("10.0.0.0/8"),
    ipaddress.IPv6Network("::1/128"),
]

# After
BLOCKED_NETWORKS: List[Union[ipaddress.IPv4Network, ipaddress.IPv6Network]] = [
    ipaddress.IPv4Network("10.0.0.0/8"),
    ipaddress.IPv6Network("::1/128"),
]
```

**Files:** web_scraper.py

### 6. Generator Return Types for Context Managers

**Problem:** @contextmanager decorated functions need Generator return type.

**Solution:**
```python
# Before
@contextmanager
def measure(self, operation: str, ...):

# After
@contextmanager
def measure(self, operation: str, ...) -> Generator[None, None, None]:
```

**Files:** performance.py

### 7. Type Narrowing with Assert

**Problem:** Optional types need to be narrowed before use in strict contexts.

**Solution:**
```python
# Before
if self._session_provided:
    return nullcontext(self.session)  # self.session is Optional[Session]

# After
if self._session_provided:
    assert self.session is not None
    return nullcontext(self.session)  # mypy knows it's Session
```

**Files:** metrics_collector.py

### 8. set[str] Type Annotations

**Problem:** Empty sets need explicit type annotations.

**Solution:**
```python
# Before
all_metric_names = set()

# After
all_metric_names: set[str] = set()
```

**Files:** metrics_collector.py, analyzer.py

### 9. Mixin Attribute Access with getattr

**Problem:** Mixins may access attributes from parent classes that aren't on the mixin.

**Solution:**
```python
# Before
service_name=self.name  # SafetyServiceMixin doesn't have 'name'

# After
service_name=getattr(self, 'name', 'unknown')
```

**Files:** service.py

### 10. Explicit Type Annotations for Optional Attributes

**Problem:** Attributes that can be None need explicit Optional[] annotation before assignment.

**Solution:**
```python
# Before
if self.templates_dir and self.templates_dir.exists():
    self.jinja_env = SandboxedEnvironment(...)
else:
    self.jinja_env = None  # Type error!

# After
self.jinja_env: Optional[SandboxedEnvironment]
if self.templates_dir and self.templates_dir.exists():
    self.jinja_env = SandboxedEnvironment(...)
else:
    self.jinja_env = None
```

**Files:** prompt_engine.py
