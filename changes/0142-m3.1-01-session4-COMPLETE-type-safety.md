## Key Fixes This Session

### 1. Missing Import Type Stubs

**Problem:** External libraries without type stubs cause import errors.

**Solution:** Add `type: ignore[import-not-found]` or `type: ignore[import-untyped]`
```python
# For libraries without stubs
import numpy as np  # type: ignore[import-not-found]
import psutil  # type: ignore[import-untyped]
```

**Files:** sequential_testing.py, analyzer.py, checkpoint_backends.py, llm_cache.py, resource_limit_policy.py, llm_providers.py

### 2. Context Manager Return Types

**Problem:** Context managers need specific return types, and __exit__ must return Literal[False] if it never returns True.

**Solution:**
```python
from typing import Literal

def __enter__(self) -> "BaseLLM":
    return self

def __exit__(
    self,
    exc_type: Optional[Type[BaseException]],
    exc_val: Optional[BaseException],
    exc_tb: Optional[TracebackType]
) -> Literal[False]:
    self.close()
    return False  # Never swallows exceptions
```

**Files:** llm_providers.py

### 3. **kwargs Annotations Throughout

**Problem:** All functions with `**kwargs` need `**kwargs: Any` for strict mode.

**Solution:**
```python
def method(self, param: str, **kwargs: Any) -> ReturnType:
    pass
```

**Files:** llm_providers.py, llm_failover.py, service.py

### 4. SQLAlchemy Query Type Ignores

**Problem:** SQLAlchemy ORM queries have complex types that mypy can't infer.

**Solution:**
```python
query = query.filter(
    Model.column == value  # type: ignore[arg-type]
)
query = query.order_by(Model.timestamp.desc())  # type: ignore[attr-defined]
```

**Files:** rollback_logger.py, service.py

### 5. Unreachable Code Type Ignores

**Problem:** Type narrowing makes some runtime checks appear unreachable to mypy.

**Solution:**
```python
if state.stage_outputs is None:
    updates["stage_outputs"] = {}  # type: ignore[unreachable]
```

**Files:** state_manager.py, secrets.py

### 6. LangGraph Complex Typing

**Problem:** Third-party LangGraph library has complex overloads and generic types.

**Solution:** Add targeted type: ignore comments
```python
def compile(self, config: Dict[str, Any]) -> StateGraph:  # type: ignore[type-arg]
    ...
    graph.add_node(name, node)  # type: ignore[call-overload]
    return graph.compile()  # type: ignore[return-value]
```

**Files:** langgraph_compiler.py, stage_compiler.py

### 7. Abstract Class Instantiation

**Problem:** Factory function instantiates abstract class (actually concrete subclass at runtime).

**Solution:**
```python
return llm_class(  # type: ignore[abstract]
    model=model,
    base_url=base_url,
    **kwargs
)
```

**Files:** llm_providers.py
